# Лабораторная работа №2

## Тема

Работа с технологиями описания исследуемой предметной области на 
примере составления интеллект-карт – mind maps.

## Цель работы

Для выбранной любой предметной области составить интеллект-карту: не 
менее 5-6 категорий на 1-ом уровне; на 2-м уровне у каждой категории 3-4 
понятия.
